MOVES Workshops and Presentations

MOVES Workshop 2011, June 14-16, 2011, Ann Arbor, Michigan.

The U.S. EPA held a three-day workshop including EPA presentations on MOVES algorithms and default data, information on ways to use MOVES more efficiently for various purposes, and discussion of ideas and plans for MOVES future development.

You will need Adobe Reader to view some of the files on this page. See EPA’s About PDF page to learn more.
  • Fleet & Activity Data in MOVES2010 (PDF)(37 pp, 330 K, June 2011)
    This presentation describes the data sources for MOVES default national vehicle population, miles travelled, and activity patterns.
  • Light-Duty Exhaust Emission Rates in MOVES2010 (PDF)(76 pp, 1 MB, June 14, 2011)
    This presentation describes source data and analysis used to develop emission rates for cars and light trucks in MOVES.
  • Updates to Energy Rates and Sourcebin Distribution in MOVES 2010a (PDF)(21 pp, 428 K, June 14, 2011)
    This presentation describes energy consumption rates in MOVES2010a and changes since MOVES2010 in how vehicles were categorized.
  • Development of Heavy-Duty Emission Rates for MOVES2010 (PDF)(46 pp, 602 K, June 14, 2011)
    This presentation describes source data and analysis used to develop emission rates for heavy trucks in MOVES.
  • The MOVES Approach to Modeling Evaporative Emissions (PDF)(27 pp, 361 K, June 13, 2011)
    This presentation describes how MOVES estimates vapor venting, liquid leaks and permeation of fuel vapors.
  • Fuel Effects & Air Toxics in MOVES2010 (PDF)(49 pp, 479 K, June 14-16, 2011)
    This presentation describes how MOVES estimates the emission effect of gasoline fuel properties and how MOVES calculates emissions of air toxics.
  • Exhaust Temperature, Air Conditioning and Inspection and Maintenance Adjustments in MOVES2010a (PDF)(60 pp, 1 MB, June 14, 2011)
    This presentation describes adjustment factors used in MOVES to estimate real world emissions.
  • MOVES Sensitivity Analysis: The Impacts of Temperature and Humidity on Emissions (PDF)(28 pp, 566 K, June 14, 2011)
    This presentation details the impact of ambient temperature and humidity on MOVES emission results.
  • MOVES Validation (PDF)(64 pp, 1 MB, June 14, 2011)
    This presentation compares MOVES results and independent estimates of emissions and fuel consumption.
  • MOVES Architecture: From the Ground Up (PDF)(21 pp, 269 K, June 2011)
    This presentation describes the structure of the MOVES2010 software.
  • MOVES Batch Computing using Workers on a Network (PDF)(26 pp, 300 K, June 15, 2011)
    This presentation describes how to automate MOVES runs and to run MOVES on multiple computers.
  • Installing and Running MOVES on Linux (PDF)(52 pp, 593 K, June 15, 2011)
    This presentation provides step-by-step instructions for converting MOVES to run on a Linux operating system.
  • Running MOVES in the Cloud (PDF)(21 pp, 486 K, June 2011)
    This presentation describes how EPA has reduced run-time for large inventories by running MOVES in parallel using commercially-available, on-demand computing.
  • SMOKE-MOVES (PDF)(34 pp, 800 K, June 2011)
    This presentation describes an EPA tool that integrates MOVES with the SMOKE emission model to produce inputs appropriate for air quality models.
  • Using the Motor Vehicle Emission Simulator (MOVES) Model in Inventory Mode to Develop Regional On-Road Emission Inputs for Air Quality Modeling Applications (PDF)(31 pp, 881 K, June 15, 2011)
    This presentation describes how MOVES was run to generate background emission inventories.
  • Application of SMOKE-MOVES Integration Tool to Regional Air Quality Modeling (PDF)(38 pp, 5 MB, June 15, 2011)
    This presentation describes how MOVES was run to generate emission inventories.
  • Using MOVES for Project-level Particulate Matter (PM) Hot-Spot Analyses (PDF)(50 pp, 2 MB, June 2011)
    This presentation provides context and a detailed example of how MOVES can be used to model emissions from a complex transportation project.
  • MOVES Training and Guidance (PDF)(18 pp, 140 K, June 2011)
    This presentation describes MOVES training courses, written guidance, and other resources for MOVES users.
  • MOVES Development Plans (PDF)(10 pp, 86 K, June 2011)
    This presentation describes EPA plans for future versions of MOVES.
  • Helping MPOs with MOVES Inputs for Transportation Conformity (PDF)(9 pp, 557 K, June 16, 2011)
    This presentation describes a variety of work with MOVES.
  • TDM2MOVES: A Travel Demand Model Application to Generate Travel Activity Input Data for the MOVES Model (PDF)(32 pp, 4 MB, June 16, 2011)
    This presentation describes a user-created tool that creates MOVES input files from travel demand model output.
  • Integration Second-by-Second Traffic Movement Data Generated from Traffic Simulation Models onto MOVES Project Level Analyses (PDF)(14 pp, 270 K, June 2011)
    This presentation describes challenges in using traffic simulation output as an input to MOVES.
  • MOVES2010a and Representative Counties (PDF)(8 pp, 279 K, June 16, 2011)
    This presentation describes a modeling approach in which MOVES results for about 200 counties were used to model the entire United States.
  • National Inventory Development with MOVES (PDF)(7 pp, 58 K, June 2011)
    This presentation describes how MOVES was run to generate national background emission inventories.
  • Comments/Requests by Susan E. Hardy, MOVES Workshop June 14-16, 2011 (PDF)(2 pp, 204 K, June 14-16, 2011)
    This is a list of one user’s suggestions for improving the MOVES model.
  • U.S. EPA's MOVES2010 vehicle emission model: overview and considerations for international application (PDF)(7 pp, 169 K, Jun 14-16, 2011)
    Presented at the International Symposium on Transport and Air Pollution, May 2010. This presentation describes how to customize MOVES to model emissions outside the United States.
